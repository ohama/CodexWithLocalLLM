# Codex vs OpenHands 벤치마크 — 따라하며 배우는 튜토리얼

이 책은 **로컬 LLM(qwen-122b) 위에서 두 코딩 에이전트 — Codex CLI 와 OpenHands — 를 같은 조건으로
비교 검증**하는 과정을, 초보 개발자가 **위에서 아래로 따라가며 직접 수행하고 그 결과를 해석**할 수
있게 정리한 것이다.

## 이 책이 답하는 질문

> "내 로컬 모델로 codex와 openhands 중 어느 쪽이, 어떤 작업에서, 얼마나 잘/빠르게 하는가?
> 그리고 그 결과를 어떻게 믿고 읽어야 하는가?"

## 큰 그림

```
두 도구(codex, openhands)
   │   같은 모델(qwen-122b) · 같은 과제 · 같은 조건
   ▼
같은 7종 과제를 시킴 (난이도 + 도메인)
   │
   ▼
성공 여부·시간·과정·규모를 자동 측정
   │
   ▼
표로 비교하고 → 올바르게 해석
```

## 이 책의 흐름 (수행 → 해석)

1. **개요** — 무엇을·왜 비교하나, 공정성의 3원칙
2. **과제 7종** — 왜 이 과제들(fib/wordstat/KV/calc/todo/csvstat/kvapi)을 골랐나 + 기대효과
3. **사전 준비** — 백엔드(qwen-122b)가 떠 있는지 확인
4. **Codex 연결** — codex를 로컬 모델에 붙이기
5. **OpenHands 연결** — openhands를 같은 모델에 붙이기
6. **수행** — 한 셀 시험 → 전체 매트릭스 실행
7. **해석** — 4가지 지표의 의미와 표 읽는 법, 흔한 오독
8. **결과 요약과 결론** — 실제 결과 + 무엇을 말할 수 있나
9. **막혔을 때** — 트러블슈팅

> **비교 대상(도구)은 codex 와 openhands 둘뿐**이고, **두 도구에 똑같이 시키는 과제가 7종(L1~L7)**
> 이다. (다른 도구는 이 책의 범위 밖.)

준비됐으면 [1. 무엇을 왜 비교하나](01-overview.md)부터 시작하자.

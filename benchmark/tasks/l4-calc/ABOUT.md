# L4 — calc (산술식 평가 CLI)

**복잡도 차원:** single-file / 알고리즘 (연산자 우선순위·괄호 파싱)
**무엇을 분별하나:** `eval` 에 의존하지 않고 **파서를 직접 구현**해 우선순위·괄호·단항 마이너스를
정확히 처리하는가.

## 결과물 (파일 리스트)

정답이 만드는 파일:

| 파일 | 역할 |
|------|------|
| `calc.py` | CLI 진입점 — 표현식 토크나이즈 + 재귀하강 파서 + 평가 + 출력 |

(단일 파일로 충분. 보조 모듈을 둬도 되지만 진입점은 반드시 `calc.py`.)

## 수행결과 (기능 및 사용법)

표현식 하나를 받아 계산 결과를 한 줄로 출력한다.

- **지원:** 정수·소수 리터럴, `+ - * /`, 단항 마이너스, 괄호, 표준 우선순위(좌결합), 공백 무시.
- **나눗셈:** 실수 나눗셈(`/`).
- **출력:** 결과가 정수값이면 소수점 없이(`14`), 아니면 `str(float)`(`2.5`). 성공 시 종료코드 0.
- **에러:** 잘못된 식·0으로 나눔·빈 식 → 종료코드 nonzero.

### 사용 예

```sh
python3 calc.py "2 + 3 * 4"      # → 14   (우선순위: 곱셈 먼저)
python3 calc.py "(2 + 3) * 4"    # → 20   (괄호 먼저)
python3 calc.py "10 / 4"         # → 2.5
python3 calc.py "-3 + 5"         # → 2    (단항 마이너스)
python3 calc.py "7 - 2 - 1"      # → 4    (좌결합)
python3 calc.py "1 / 0"          # → (에러) 종료코드 nonzero
```
